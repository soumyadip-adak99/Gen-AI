# Python Control Flow Practice

This project contains a Jupyter Notebook (`control_flow_assignment.ipynb`) with several tasks demonstrating Python's control flow structures.

## Tasks Overview

*   **Task 1: Discount Rules:** Uses `if/elif/else` to apply discounts to an order amount.
*   **Task 2: Process Multiple Orders:** Uses a `for` loop to process a list of orders.
*   **Task 3: User Menu:** Uses a `while` loop to create an interactive menu for order entry.
*   **Task 4: Loop Control:** Uses `break` and `continue` to handle special conditions while processing sales data.

## How to Run

The easiest way to run this code is to open the `control_flow_assignment.ipynb` file in a Jupyter environment (like Jupyter Notebook, JupyterLab, or VS Code) and run the cells for each task individually.
