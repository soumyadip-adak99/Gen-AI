# Assignment 1: Python Data Structures

This assignment covers fundamental Python data structures.

## Tasks

### Task 1: Product Collection (Lists & Tuples)
- Create and manipulate a list of products.
- Work with tuples representing product information.

### Task 2: Categories (Sets)
- Use sets to store and manage unique product categories.

### Task 3: Product Pricing (Dictionaries)
- Use dictionaries to store and manage product prices.
- Perform operations like adding, updating, and removing items.
- Calculate the average price of products.

### Task 4: Combination Operations
- Combine different data structures to create a product catalog.
- Group products by category.
